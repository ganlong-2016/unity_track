# 卡车 3D 模型使用说明

## 创建卡车

1. 在 Unity 菜单选择：**GameObject → 3D Truck → Create Truck**
2. 场景中会生成一个名为 `Truck` 的根物体，包含驾驶室、货厢、车门、车窗和车轮。

## 结构说明

- **Truck**（根）
  - **Cab**（驾驶室）
    - CabFrame：驾驶室主体
    - LeftDoor / RightDoor：左/右车门（可绕铰链旋转）
    - FrontWindow：前挡风玻璃
    - LeftFrontWindow / RightFrontWindow：左/右前侧窗
    - LeftRearWindow / RightRearWindow：左/右后侧窗（可下滑“打开”）
  - **CargoBed**（货厢）
  - **Wheels**（四个车轮）

## 车门与车窗控制

根物体上挂有 **TruckVehicleController** 组件：

- **Q**：开关左车门  
- **E**：开关右车门  
- **W**：开关全部车窗（下滑/上滑）

可在 Inspector 中调整：

- 车门打开角度、动画时长  
- 车窗打开位移、动画时长  
- 勾选/取消 **Use Key Input** 以启用或禁用按键控制  

脚本接口（可在其他脚本中调用）：

- `SetLeftDoorOpen(bool)` / `SetRightDoorOpen(bool)`  
- `ToggleLeftDoor()` / `ToggleRightDoor()`  
- `SetAllWindowsOpen(bool)` / `ToggleAllWindows()`

## 保存为预制体

1. 选中场景中的 **Truck** 根物体  
2. 菜单：**GameObject → 3D Truck → Save Truck as Prefab**  
3. 预制体会保存到 `Assets/Prefabs/Truck.prefab`，之后可拖入场景复用。

## 替换为自定义模型

若使用 Blender/Maya 等导出的卡车模型：

1. 确保模型层级中包含名为 `LeftDoor`、`RightDoor` 等的子物体（或与脚本中查找名称一致）。  
2. 将 **TruckVehicleController** 挂到卡车根物体上。  
3. 在 Inspector 中手动指定 **Left Door**、**Right Door** 和 **Windows** 数组，或保持默认并调用一次 `CollectParts()`（例如在 `Start` 里），让脚本按名称自动绑定。

这样即可保留“开门、开窗”等通用车模功能。
